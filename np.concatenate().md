`numpy`提供了`numpy.concatenate((a1,a2,...), axis=0)`函数。能够一次完成多个数组的拼接。其中a1,a2,...是数组类型的参数

```python
a = np.array([1,2,3])
b = np.array([11,22,33])
c = np.array([44,55,66])
np.concatenate((a,b,c) , axis = 0) #对于一维数组，axis = 0  或 1 效果一样 都是叠加在一起
#  array([ 1, 2, 3, 11, 22, 33, 44, 55, 66])

#当对于二维数组时
a=np.array([[1,2,3],[4,5,6]])
b=np.array([[11,21,31],[7,8,9]])
np.concatenate((a,b),axis=0)  
#axis = 0 表示对应列的数组进行拼接
#array([[ 1,  2,  3],  [ 4,  5,  6],  [11, 21, 31],   [ 7,  8,  9]]) 
np.concatenate((a,b),axis=1) 
#axis=1表示对应行的数组进行拼接
#array([[ 1,  2,  3, 11, 21, 31], [ 4,  5,  6,  7,  8,  9]])
```
