#### get_ellipse
```matlab
function [X,Y]=get_ellipse(Mu,Sigma,S,pntNum)

% (X-Mu)*inv(Sigma)*(X-Mu)=S

invSig=inv(Sigma);

[V,D]=eig(invSig);

aa=sqrt(S/D(1));

bb=sqrt(S/D(4));

t=linspace(0,2*pi,pntNum);

XY=V*[aa*cos(t);bb*sin(t)];

X=(XY(1,:)+Mu(1))';

Y=(XY(2,:)+Mu(2))';

end
```
